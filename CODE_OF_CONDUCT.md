# Code Of Conduct

This project follows a simple standard: be respectful, constructive, and kind.

## Our Standards

Examples of positive behavior:

- Being respectful in discussions
- Giving actionable feedback
- Assuming good intent
- Helping others learn

Examples of unacceptable behavior:

- Harassment or personal attacks
- Trolling or insulting language
- Publishing private information without permission
- Disruptive or hostile behavior

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, or pull requests that do not align with this code of conduct.

## Scope

This applies to project spaces such as issues, pull requests, and discussions.
